# すみかみらい サイト制作ルール

- 正式名称は「すみかみらい 一級建築士事務所」、屋号は「すみかみらい」とする。
- HTML・CSS・Vanilla JavaScriptで制作し、GitHub Pagesで公開できる相対パスを使う。
- 既存の文章・画像・ロゴは確認なしに削除、変形、再生成しない。
- 未確定の所在地、連絡先、営業時間、登録番号、料金、実績、事例、提携先などは捏造しない。`TODO: 公開前に正式情報へ差し替え` と明記する。
- 一級建築士事務所登録前に登録済みと誤認させない。宅地建物取引士の資格と宅建業者としての仲介を混同しない。解体工事を直接施工する表現も避ける。
- 不安を煽らず、工事や売却を急がせない。「選択肢を整理する」姿勢を中心にする。
- 一般の方、特に60代以上でも読みやすい日本語、16px以上の本文、十分なコントラストとタップ領域を保つ。
- セマンティックHTML、見出し階層、代替テキスト、キーボード操作、フォーカス表示、フォームラベル、文章によるエラー、スキップリンク、`prefers-reduced-motion` に対応する。
- ネイビー、白、落ち着いた緑をCSS変数で管理する。派手な装飾、過剰な影・角丸・アイコン・アニメーションを避ける。
- 架空の人物写真、施工例、お客様の声、口コミ、数字実績を作らない。
- 問い合わせ送信先が確定するまでは、入力・確認画面までとし外部送信しない。
